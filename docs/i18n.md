# Multi-language Support

## Supported Languages

| Language | Code | Default |
|----------|------|---------|
| 简体中文 | `zh-CN` | Yes |
| English (US) | `en-US` | No |

## Switching Language

### Via Command Line

```bash
# Chinese (default)
dnm --lang zh-CN

# English
dnm --lang en-US
```

### Via Interactive Mode

When using `dnm -i`, the first prompt asks you to select a language.

## Language Scope

The language setting affects:

- Console output messages
- Help text (`dnm --help`)
- Interactive prompts
- Progress bar labels
- Safe mode prompts

It does **not** affect:

- File paths
- Error messages from the OS
- Log file format (always uses ISO timestamps)

## Contributing Translations

To add a new language:

1. Add translations to `lib/i18n.js`
2. Follow the existing key structure
3. Submit a PR

See [Contributing Guide](/contributing) for details.
