# 安全模式

## 概述

安全模式**默认启用**。它确保你在任何删除操作之前都能看到将要删除的内容。

## 工作流程

1. `dnm` 递归扫描目标目录，查找所有 `node_modules`
2. 显示找到的目录编号列表
3. 你选择要删除哪些
4. 只有被选中的目录会被删除

## 选择语法

| 输入 | 操作 |
|------|------|
| *(直接回车)* | 删除全部 |
| `3` | 只删除第3个 |
| `1-5` | 删除第1到第5个 |
| `1,3,5` | 删除第1、3、5个 |
| `1-3,7,9-12` | 组合选择 |
| `q` 或 `Q` | 取消操作 |

## 示例

```bash
$ dnm -s

═══════════════════════════════════════════════════
  安全模式 - 发现的 node_modules 目录
═══════════════════════════════════════════════════

[1] /projects/app1/node_modules (150.5 MB)
[2] /projects/app2/node_modules (89.2 MB)
[3] /projects/lib/node_modules (45.8 MB)
[4] /projects/demo/node_modules (12.3 MB)

输入要删除的序号: 1,3

已选择删除 2 个目录
```

## 禁用安全模式

如果你想跳过确认步骤：

```bash
dnm --no-safe
```

> ⚠️ 这将立即删除所有找到的 `node_modules` 目录。请确保已备份重要项目。
