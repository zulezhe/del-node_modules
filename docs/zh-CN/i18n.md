# 多语言支持

## 支持的语言

| 语言 | 代码 | 默认 |
|------|------|------|
| 简体中文 | `zh-CN` | 是 |
| English (US) | `en-US` | 否 |

## 切换语言

### 通过命令行

```bash
# 中文（默认）
dnm --lang zh-CN

# 英文
dnm --lang en-US
```

### 通过交互模式

使用 `dnm -i` 时，第一个提示就是语言选择。

## 语言作用范围

语言设置影响：

- 控制台输出消息
- 帮助文本（`dnm --help`）
- 交互式提示
- 进度条标签
- 安全模式提示

**不影响**：

- 文件路径
- 操作系统错误消息
- 日志文件格式（始终使用 ISO 时间戳）

## 贡献翻译

添加新语言：

1. 在 `lib/i18n.js` 中添加翻译
2. 遵循现有的键结构
3. 提交 PR

详见 [贡献指南](/zh-CN/contributing)。
