# 交互模式

交互模式提供逐步引导的向导来配置和运行 `dnm`。推荐首次使用的用户使用。

## 启动交互模式

```bash
dnm -i
# 或
dnm --interactive
```

## 向导步骤

### 1. 语言选择

```
? 选择语言 / Select language:
  ❯ 简体中文 (Simplified Chinese)
    English (US)
```

### 2. 目标目录

```
? 输入要扫描的目录路径: /home/user/projects
```

### 3. 忽略目录

```
? 输入要忽略的目录（逗号分隔，留空表示不忽略）:
```

### 4. 安全模式

```
? 启用安全模式（删除前显示列表并确认）？(Y/n)
```

### 5. 显示大小

```
? 计算并显示目录大小？(y/N)
```

### 6. 进度条

```
? 显示进度条？(Y/n)
```

### 7. 日志级别

```
? 选择日志级别:
  debug
  ❯ info
    warn
    error
```

### 8. 日志文件

```
? 将日志保存到文件？(y/N)
```

如果选择是：
```
? 输入日志文件路径: cleanup.log
```

### 9. 最终确认

```
⚠️  这将删除所有 node_modules 目录。是否继续？(y/N)
```

## 适用场景

- 首次使用 `dnm`
- 不确定命令行选项
- 想要引导式体验
